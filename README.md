# CRUD de Contatos – PHP + MySQL

Projeto completo desenvolvido para estudo e portfólio.

## 🔧 Tecnologias usadas
- PHP
- MySQL
- HTML
- CSS
- JavaScript (mínimo)
- XAMPP
- Git & GitHub

## 📌 Funcionalidades
- Login com sessão
- Listagem de contatos
- Adicionar contato
- Editar contato
- Excluir contato com confirmação
- Busca por nome
- Ordenação por nome, email e telefone
- Layout responsivo
- Botões estilizados
- Proteção de páginas

## 🚀 Como executar
1. Clonar o repositório
2. Colocar dentro de `htdocs`
3. Criar banco `crud_contatos`
4. Importar tabela `contatos` e `usuarios`
5. Acessar `http://localhost/crud_contatos/login.php`
